## English (Alternative)

---

**I had 47 tabs open. Now I have zero.**

You know the feeling. Dozens of browser tabs you're afraid to close. Articles for "later." Tools you want to remember. Research you might need someday.

So they stay open. For weeks. Your browser gets slower, your mind more cluttered.

I built a tool to fix this:

**TabTriage** captures all tabs in a browser window with one click, uses AI to summarize and cluster them by topic, then gives you a fast triage interface to decide: read later, archive, route to Notion — or just close. Guilt-free.

How it works:
1. One click captures every tab
2. Mozilla Readability extracts full page content
3. Claude summarizes each page in 2-3 sentences
4. Tabs auto-cluster by topic
5. Quick decisions in a clean UI
6. Close triaged tabs directly

Everything stays local. No cloud service. No data shared. Open source (MIT).

Built with Python, FastAPI, SQLite, and Claude. In a weekend.

https://github.com/marcomaas/tabtriage
